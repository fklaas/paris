TEILNEHMER-CENTER UND LIVE-KARTE

1. In Supabase den SQL Editor öffnen.
2. Die Datei PARIS-PEOPLE-LIVE-MAP-SETUP.sql vollständig ausführen.
3. Danach alle Dateien dieser ZIP auf den Webspace hochladen.
4. Die App einmal vollständig neu laden. Bei einer installierten PWA gegebenenfalls schließen und erneut öffnen.

NEU
- synchronisierbare Profilbilder oder Initialen mit Avatarfarbe
- Rollen, Online-/Offline-Status, letzter Sync und Gerätetyp
- aktuelle Aktivität jedes Teilnehmers
- Aktivitätsfeed für Fotos, Live Moments und besuchte App-Bereiche
- freiwillige Standortfreigabe pro Profil
- eigener Profilpunkt Live-Karte
- Leaflet-Karte mit OpenStreetMap-Kartenmaterial
- Entfernung zwischen zwei sichtbaren Teilnehmern
- Teilnehmer direkt erneut einladen

DATENSCHUTZ
Die Live-Karte zeigt einen Standort nur, wenn die betreffende Person die Freigabe in ihrem Profil oder auf der Live-Karte ausdrücklich aktiviert hat. Beim Ausschalten bleibt die zuletzt gespeicherte Position technisch in Supabase, wird anderen Teilnehmern aber nicht mehr ausgeliefert. Eine spätere Version kann zusätzlich eine Funktion zum sofortigen Löschen der letzten Position erhalten.
