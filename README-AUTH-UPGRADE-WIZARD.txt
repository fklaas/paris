PARIS AUTH – KONTO SICHERN V2

Der anonyme Account wird jetzt in drei getrennten Schritten umgewandelt:
1. E-Mail und Profildaten verknüpfen.
2. E-Mail über den zugesandten Link bestätigen.
3. Erst danach ein Passwort festlegen.

Wichtig:
- Anonymous Sign-ins vorerst aktiviert lassen.
- Allow manual linking muss in Supabase aktiviert sein.
- Die Site URL und Redirect URL müssen auf die GitHub-Pages-App zeigen.
- Keine neue SQL-Ausführung erforderlich.
