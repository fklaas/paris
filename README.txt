GitHub Pages:
1. ZIP entpacken.
2. Alle enthaltenen Dateien gemeinsam in das Hauptverzeichnis des Repositories hochladen.
3. Vorhandene Dateien ersetzen.
4. Alte Dateien menu-elio-1.jpg bis menu-elio-4.jpg können gelöscht werden.
5. Die neuen Dateien menu-elio-1.webp bis menu-elio-4.webp müssen hochgeladen werden.
6. Nach dem Commit Strg+F5 drücken.
7. Bei einer installierten PWA die App vollständig schließen und erneut öffnen.

Der Speisekarten-Viewer zeigt jede Seite vollständig an.
Zoom ist per +/-, Tastatur, Strg/Cmd + Mausrad und auf dem iPhone per Zwei-Finger-Geste möglich.
