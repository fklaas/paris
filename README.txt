GitHub Pages:
1. ZIP entpacken.
2. Alle Dateien gemeinsam in das Hauptverzeichnis des Repositories hochladen.
3. Vorhandene Dateien ersetzen.
4. Nach dem Commit Strg+F5 drücken.
5. Bei einer installierten PWA die App vollständig schließen und erneut öffnen.

Korrektur:
Jede Speisekartenseite wird nun immer vollständig in das Popup eingepasst.
Es gibt keinen Seitenbreite-Modus und kein vertikales Abschneiden mehr.
